# Travessia dos Missionários e Canibais
A incrível jornada de Frei Sardinha, seus cardeais, enfrentando um violento rio, na companhia dos índios Caetés.

Visualização online exportada:  
https://ali1ariel.github.io/travessia

## Como executar localmente

1. Instale o Jupyter:
```bash
pip install notebook
```

2. clone esse repositório e abra o arquivo 'travessia.ipynb' por meio da interface do jupyter.
